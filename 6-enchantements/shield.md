# Shield

## Informations
Un bouclier Antiémeute ! L'enchantement Shield est fusionnable avec une [pièce d'armure](https://histeria.fr/wiki/2-equipement).
Quand le porteur est frappé, tous les dégâts du coup reçu s'annulent !

**Niveau maximum :** 3
